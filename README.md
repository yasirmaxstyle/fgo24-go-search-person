# SEARCHING PEOPLE BY NAME PROGRAM
This program check whether the name by input match any names in the list, if any it will return names in slice data types, otherwise it returns empty slice.

## How to run this program
### Prerequisite
- [Go Programming Language](https://go.dev/)

### Manual

1. Clone this repository
```bash
git clone https://github.com/yasirmaxstyle/fgo24-go-search-person.git
```
2. Get into the path
```bash
cd fgo24-go-search-person
```
3. Run the program
```bash
go run main.go
```

## Technologies
- [Go Programming Language](https://go.dev/)
